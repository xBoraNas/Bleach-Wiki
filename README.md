# ⚔️ Bleach-Wiki

A comprehensive fan wiki website dedicated to the Bleach anime and manga universe, built using HTML, CSS, and JavaScript. This project aims to present the different dimensions, characters, and storylines of the Bleach world in an interactive and visually appealing way.

## 🌐 Live Preview
You can access the live version of the website via the link below:
👉 [Bleach Wiki Live Site](https://xboranas.github.io/Bleach-Wiki/)

## 🚀 Features
- **Multi-Page Structure:** Dedicated pages designed for different worlds in the universe (`index.html`, `huecomundo.html`, `soulsociety.html`, `humanworld.html`).
- **Interactive Form:** A customized `form.html` page for user feedback and engagement.
- **Multimedia Integration:** Visual and video content (`videos/`, `images/`) to showcase characters and iconic scenes.
- **Dynamic Content:** Integrated JavaScript (`js/`) scripts to enhance the overall user experience.
- **Thematic UI/UX:** A modern, fully CSS-styled dark theme tailored to match the atmospheric vibe of Bleach.

## 🛠️ Built With
- **HTML5:** Semantic structure and layout design.
- **CSS3:** Custom styling, color palettes, layouts (Flexbox/Grid), and visual effects.
- **JavaScript (ES6+):** In-page dynamic interactions and core functionalities.

## 📂 Project Structure
```text
├── css/            # Style files (Layouts and visual themes)
├── images/         # Character art, background images, and logos
├── js/             # JavaScript source files (Dynamic features)
├── videos/         # Anime clips and video content
├── index.html      # Homepage (Overview and featured characters)
├── huecomundo.html # Hueco Mundo world and the Arrancars
├── soulsociety.html# Soul Society and the Gotei 13 captains
├── humanworld.html # Human World and Karakura Town
├── form.html       # Contact and feedback form
└── README.md       # Project documentation file
```

## 💻 Local Setup and Installation
To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/xBoraNas/Bleach-Wiki.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bleach-Wiki
   ```
3. Open `index.html` by double-clicking it in any web browser (Chrome, Edge, Safari, etc.).
